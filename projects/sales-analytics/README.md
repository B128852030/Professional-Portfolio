# Sales Analytics Project

This project demonstrates **Python, SQL, and Data Visualization** skills using a sample sales dataset.

## Project Overview
- Load sales data from CSV into SQLite database
- Run SQL queries to analyze:
  - Total revenue per product
  - Top customers by spending
  - Sales by region
  - Monthly sales trends
- Visualize results using matplotlib:
  - Bar chart for product revenue
  - Pie chart for regional revenue
  - Line chart for monthly sales

## Technologies Used
- Python 3.11
- pandas
- sqlite3
- matplotlib
